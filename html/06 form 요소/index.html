form요소

웹브라우저를 이용해 사용자가 제아할 수 있는 ui

form요소의 개념

  • 전송할 데이터의 범위
  • action속성 : 데이터를 전송할 주소
  • method속성
    • get : 주소창을 이용해 텍스트로 테이터를 전송
    • post : 보안이 필요한 데이터, 바이너리파일을 전송

form의 구획요소

  • fieldset : 관련된 데이터의 범위를 설정
  • legend : fieldset의 제목

form의 접근성

  • label요소 : for와 id속성으로 연결
  • title속성 : label 텍스트를 대체할 수 있는 속성

input 요소와 type

  • text : default속성

입력양식의 종류

  • input
  • select,option
  • textarea
  • button
